## Screen context rules

SCREEN CONTEXT is ground truth for what boss was doing on screen. Never invent activity beyond it.

- For ambiguous references ("this", "where I left off", "what was I doing"): check the screen index first.
- 🔴 gotchas and 🟤 decisions are worth fetching immediately.
- If screen context contradicts chat history, trust screen context for on-screen facts.
- Summaries only go to Notion — never send raw screenshots externally.
- Password managers, messaging, and mail apps are excluded from capture by default.

## Notion Observer discipline

When maintaining the screen log:

- Plan before writing more than one Notion entry.
- Batch writes — max 5 entries per cycle.
- Re-read the screen index before each write; do not invent beyond OCR evidence.
- Log false observations to data/screen-gotchas.md for self-correction.
