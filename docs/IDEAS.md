# Hackathon idea candidates

Score each 1–5 at kickoff. Pick **one** idea — one working journey beats five half-features.

## Tier A — strongest for our team (full-stack + mobile + automation)

### 1. Voice Lead Qualifier
- **Flow:** Apify scrapes Google Maps leads → n8n enriches → ElevenLabs voice agent calls/leaves voicemail → web dashboard
- **Tracks:** ElevenLabs, Apify, n8n, Overall
- **Demo hook:** Live phone call in pitch room

### 2. Multilingual City Assistant (The Hague)
- **Flow:** Mobile app → Apify scrapes local events/transport → ElevenLabs voice answers in NL/EN/AR
- **Tracks:** ElevenLabs, Crowd Favorite
- **Demo hook:** Local relevance + mobile polish

### 3. Competitor Watchdog
- **Flow:** Apify monitors prices/ads/reviews → n8n alerts → dashboard shows diffs
- **Tracks:** Apify, n8n, Tech Winner
- **Demo hook:** Live data change on screen

## Tier B — backend-heavy

### 4. Micro-SaaS with Fluxzero
- **Flow:** Narrow vertical SaaS (booking, approvals) with real auth via Fluxzero
- **Tracks:** Fluxzero, Overall

### 5. Document-to-Action Pipeline
- **Flow:** Upload PDF → n8n extracts → actions + ElevenLabs audio summary
- **Tracks:** n8n, ElevenLabs

## Tier C — wildcard

### 6. AI Audio Briefing
- **Flow:** Apify news scrape → LLM summary → ElevenLabs podcast → player
- **Tracks:** ElevenLabs
